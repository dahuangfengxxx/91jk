# 随息居 · 智能食疗检索平台

## 项目简介

随息居是一个基于中医食疗理论的智能检索平台，为用户提供专业的食材搭配建议和食疗方案。

## 主要功能

- 智能食材检索
- 食疗配方推荐
- 食材属性分析
- 个性化膳食建议

## 核心文件

- `index.html` - 主要网站界面
- `index_new.html` - 新版本界面
- `ingredients_master.csv` - 食材主数据库
- `recipes_master.csv` - 配方数据库
- `recipe_ingredients_master.csv` - 配方食材关联数据
- `assets/` - 网站静态资源目录

## 使用方法

直接在浏览器中打开 `index.html` 或 `index_new.html` 即可使用。

## 数据说明

项目包含完整的中医食材数据库和配方数据库，支持多维度检索和智能推荐。
